### Commit Message Prefix

| **Prefix**      | **설명**                                                   |
|------------------|-----------------------------------------------------------|
| **`feat`**      | 새로운 기능 추가                                           |
| **`fix`**       | 버그 수정                                                 |
| **`refactor`**  | 코드 리팩토링                            |
| **`build`**     | 빌드 시스템, 의존성 추가/수정                              |
| **`chore`**     | 코드 변경 외의 자잘한 작업 (빌드 파일, 설정 변경 등)        | 
| **`docs`**      | 문서 수정                                                 |
| **`test`**      | 테스트 코드 추가/수정                                      | 
| **`perf`**      | 성능 개선                                                 | 

### 기술 스택
- Spring Boot 3.3.5
- Java 17
- Spring Security 6.3.4
- JJWT
- AWS S3
- AWS CodeDeploy
- AWS EC2
- MySQL
- Github Actions
